# c9.io app

This is created to use cloud 9 service as a single application.

```bash
$ npm install -g electron electron-packager
$ npm install
$ electron-packager . c9-app --platform=win32 --arch=x64
```

Change directory to 'c9-app' and execute 'c9-app.exe'
